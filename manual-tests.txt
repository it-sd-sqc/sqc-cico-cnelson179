Module 4: Ticket: 402
Title: Card number input should be limited to 8 digits

Steps:
1. Launch the CiCo application.
2. Click inside the card number input field.
3. Type: 12345678
4. Verify that all 8 digits appear in the field.
5. Attempt to type another digit: 9
6. Clear the field.
7. Type: 123456789

Expected Result:
The field should only contain 12345678.
The 9th digit should not be accepted.
Only the first 8 digits should appear.

Ticket 402 – Card Number Length Validation

Steps:
1. Launch the CiCo application.
2. Click in the card number input field.
3. Type a number longer than 8 digits (example: 123456789).
4. Observe the behavior of the text field.

Expected Result:
The field should prevent input beyond 8 digits.
Only numeric characters should be accepted.

Actual Result:
The field stops accepting characters after 8 digits and ignores invalid input.


Module 5 Ticket: Ticket 502 - Close application when window closes

Steps:
1. Run the application.
2. Click the window close button (X).

Expected Result:
The application fully exits and the console prints "Clean shutdown".

Ticket 502 Manual Test

1. Run the application.
2. The CiCo window should appear.
3. Close the window using the X button.
4. Verify the application completely exits.
5. Verify the terminal shows "Clean shutdown".

Expected Result:
Application closes completely when the window is closed.
